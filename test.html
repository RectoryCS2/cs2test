Steps after CDing into a cloned folder: Part 1) Make sure Master branch is all set This means: All changes added All changes committed (with message) All changes pushed to Github Once that's done... Part 2) Make a branch This means... git checkout -b YourBranchName Part 3) Make changes to the code ON YOUR BRANCH Part 4) Add, commit, and push your changes This means: git add -A git commit -m "your note about these changes" git push origin YourBranchName
